WooCommerce FreshBooks Integration plugin allows you to sync your WooCommerce orders with FreshBooks account.

== Description ==

**What is WooCommerce FreshBooks Integration?**

FreshBooks is an accounting service. In other words, it is a service where you can manage the invoices you issue to your customers and your company budget.

Our plugin uses the FreshBooks API so you can automatically invoice your customers during WooCommerce orders.

In this way, you can easily issue automatic invoices for users and manage invoices.

See: **<a href="https://github.com/BeycanPress/woocommerce-freshbooks" target="_blank">Documentation</a>**

= Privacy notices =

With the default configuration, this plugin, in itself, does not:

* use cookies;
* track users by stealth;
* send any data to external servers;

== Installation ==

1. Upload the entire `/woocommerce-freshbooks` directory to the `/wp-content/plugins/` directory.
2. Activate WooCommerce FreshBooks Integration through the 'Plugins' menu in WordPress.
3. Enjoy.
